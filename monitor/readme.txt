How to run the system:

    1) Run the server:

        python boun_server.py [-p PORT]

    2) Run the client:

        python boun_client.py [OPTIONS]

* The boun_server will start to listen port PORT for commands from the boun_client.

* The boun_client has many options. Run boun_client with -h option for help.
